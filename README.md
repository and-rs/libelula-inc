### AI Categorizer

1. Install uv (please install globally)

```
pip install uv
```

3. install dependencies

```
uv pip install pyproject.toml
```

or

```
uv sync
```

2. Start the app locally

```
uv run fastapi dev
```

3. Start the TailwindCSS CLI to update css in realtime

```
uv run twcss.py
```
